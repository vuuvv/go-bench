# Go Plus

Go Plus is a VSCode extension for improving Go table-driven test workflows.

Milestone 0 provides the extension skeleton: TypeScript build, lint/test commands, Go-related activation events, a basic output channel, and a no-op command.

## Development

```sh
npm install
npm run compile
npm run lint
npm test
```

Use the `Run Go Plus Extension` launch configuration to start an Extension Development Host.
